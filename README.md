זוהי אפליקציית terminal שמציגה תמונת מצב של תור כניסה לpit stop.

האפליקציה מאפשרת לראות מי כבר נכנס וסיים, מי עדיין מחכה, ומי הבא בתור — ומאפשרת גם "לשלוח הודעת רדיו" מדומה לנהג הבא, וגם לחפש רכב ספציפי לפי המספר שלו.



הרצת האפליקציה:

~~~
node main.js
~~~


מבנה הקבצים:

main - להרצת האפליקציה

data/raceData.json - לשמירת הנתונים

services/service.js - לעיבוד הנתונים

raceService.js - לקריאת הנתונים


האפליקציה מדגימה:

תמונת מצב נוכחית של המירוץ - 
כולל שם המירוץ, מספר סיבוב, סך כל המכוניות במירוץ.

תמונת מצב נוכחית של ה pit stop - 
 מי כבר נכנס וסיים, מי עדיין מחכה, ומי הבא בתור — ומאפשרת גם "לשלוח הודעת רדיו" מדומה לנהג הבא, וגם לחפש רכב ספציפי לפי המספר שלו.
